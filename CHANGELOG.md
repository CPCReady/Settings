# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-22

### Added
- Initial release of CPCReady Config
- GUI configuration interface using PySide6
- Support for configuring drives (A and B)
- Support for multiple emulators (RetroVirtualMachine, M4Board, CPCEmu)
- CPC system settings (model, video mode, user number)
- TOML-based configuration storage
- Cross-platform support (Windows, macOS, Linux)
- Custom application icon
- GitHub Actions workflow for automated builds

### Features
- Modern Qt-based interface with tabbed navigation
- File browser for selecting DSK files and emulator executables
- IP address validation for M4Board configuration
- Configuration reload functionality
- macOS Dock icon support using pyobjc-framework-Cocoa
